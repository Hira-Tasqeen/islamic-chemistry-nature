## üìñ About

This website showcases the groundbreaking achievements of Muslim scholars who laid the foundations of modern chemistry and natural sciences. During the Islamic Golden Age, these brilliant minds transformed alchemy into systematic science, developed experimental methods, and made discoveries that are still relevant today.

### Featured Scholars

- **Jabir ibn Hayyan (721-815 CE)** - Father of Chemistry
- **Al-Razi (854-925 CE)** - Pioneer of Medical Chemistry
- **Ibn Sina (980-1037 CE)** - The Prince of Physicians
- **Al-Biruni (973-1048 CE)** - Master of Precise Measurement
- **Al-Kindi (801-873 CE)** - Philosopher of the Arabs
- **Al-Jazari (1136-1206 CE)** - Father of Robotics & Engineering

### Key Discoveries Highlighted

- Laboratory equipment and apparatus
- Distillation techniques
- Strong acids (sulfuric, nitric, aqua regia)
- Crystallization methods
- Pharmaceutical chemistry
- Classification systems
- Experimental scientific method
- Precise measurement techniques

## üåê Getting Started

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/Hira-Tasqeen/islamic-chemistry-nature.git
   cd islamic-chemistry-nature
   ```

2. **Open in browser**
   
   Simply open `index.html` in your web browser:
   ```bash
   # On Linux/Mac
   xdg-open index.html  # or 'open index.html' on Mac
   
   # On Windows
   start index.html
   ```

   Or use a local server (recommended):
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (if you have 'http-server' installed)
   npx http-server
   ```

3. **View in browser**
   
   Navigate to `http://localhost:8000`

### GitHub Pages Deployment

1. Go to your repository settings on GitHub
2. Navigate to "Pages" section
3. Select the `main` branch as source
4. Click "Save"
5. Your site will be available at `https://hira-tasqeen.github.io/islamic-chemistry-nature/`

### Alternative Deployment Options

- **Netlify**: Drag and drop deployment
- **Vercel**: Connect your GitHub repository
- **Cloudflare Pages**: Deploy directly from Git
